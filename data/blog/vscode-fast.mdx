# 关于vscode的快捷键

## 扩展选择：

```js
Shift + Alt + Right Arrow // Windows/Linux
Ctrl + Shift + Cmd + Right Arrow // macOS
```

## 缩小选择：

```js
Shift + Alt + Left Arrow // Windows/Linux
Ctrl + Shift + Cmd + Left Arrow // macOS
```

## 选择当前行：

```js
Ctrl + L // Windows/Linux
Cmd + L // macOS
```

## 选择所有出现的当前选择：

```js
Ctrl + Shift + L // Windows/Linux
Cmd + Shift + L // macOS
```

## 选择下一个匹配项：

```js
Ctrl + D // Windows/Linux
Cmd + D // macOS
```

## 选择从光标到行尾：

```js
Shift + End // Windows/Linux
Shift + Cmd + Right Arrow // macOS
```

## 选择从光标到行首：

```js
Shift + Home // Windows/Linux
Shift + Cmd + Left Arrow // macOS
```

## 选择整个文档：

```js
Ctrl + A // Windows/Linux
Cmd + A // macOS
```
