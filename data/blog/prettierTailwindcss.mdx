# 安装插件

```
npm install prettier prettier-plugin-tailwindcss -D
```

-D 表示 devDependencies 下的依赖，即生产环境不会用到的库

# 配置文件

创建一个 prettier.config.cjs 文件作为项目的 prettier 配置文件

```javascript
module.exports = {
  plugins: ['prettier-plugin-tailwindcss'],
  semi: false,
  tabWidth: 4,
  arrowParens: 'avoid',
  printWidth: 800,
  trailingComma: 'none',
}
```
