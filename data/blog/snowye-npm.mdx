# snowye-npm

npm 包的功能是比较简单的主要就是 **_exec_** 用来执行终端的一些命令

那么就介绍一下开发的工具吧

- 命令行开发工具 commander
- 终端文字美化工具 chalk
- 命令行输入库 inquirer
- 命令行进度条库 cli-progress
- npm 包构建工具 father

> npm 搜索 snowye 下载 或者 npx snowye 可直接使用（推荐）
